## About the Project

This is a modern, mobile-friendly website focused on helping users explore and stay motivated while learning a new language.  
It features a strong hero/landing section and structured content sections that highlight benefits, learning tips, and an easy-to-follow layout — making it a great beginner-friendly front-end project.

✅ Built with **HTML & CSS**  
✅ Fully responsive and beginner-friendly  
✅ Great practice for layout, spacing, and clean UI structure
